# Security Policy

## Supported Versions

Only the latest release of Ignite receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not open a public GitHub issue.**
2. Email the details to **contact.promethys@gmail.com**.
3. Include a clear description of the issue, the steps to reproduce it, and the
   potential impact.

## Expected Response

- We will acknowledge your report within **48 hours**.
- We will investigate and provide an estimated timeline for a fix based on
   severity.
- Once the issue is resolved we will credit responsible disclosure unless you
   prefer to remain anonymous.

Thank you for helping keep Ignite and its users safe.
